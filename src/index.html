Este ejercicio de programación es puramente interactivo. Está basado en el cifrado César, uno de los primeros métodos de cifrado conocidos de la antigua Roma. Julio César lo usó para enviar órdenes a sus generales en los campos de batalla. Consistía en escribir el mensaje con un alfabeto latino al cual se le apliacaba un algoritmo. En aquella época, el receptor del mensaje ya conocía el código secreto para descubrir el mensaje cifrado. Para este ejercicio, no hay una clave o un código secreto único o específico, sólo debe cumplir dos condiciones: debe ser un número (de uno o de varios dígitos, positivo o negativo) y para cifrar y descifrar correctamente el mensaje, tanto el remitente como el destinatario deben conocerlo.
